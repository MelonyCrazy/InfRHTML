Урок 1: Введение в HTML

HTML (HyperText Markup Language) - это основной язык для создания веб-страниц. Он определяет структуру веб-страницы с помощью различных тегов.

Пример простейшего HTML-документа:

<!DOCTYPE html>
<html>
<head>
    <title>Пример страницы</title>
</head>
<body>
    <h1>Привет, мир!</h1>
</body>
</html>

Урок 2: Основные теги

HTML использует теги для обозначения различных частей документа. Вот некоторые из основных тегов:

  • <h1> - <h6>: Заголовки различных уровней
  • <p>: Абзац текста
  • <a>: Ссылка
  • <img>: Изображение

Урок 3: Списки и ссылки

В HTML можно создавать упорядоченные и неупорядоченные списки. Вот пример:

<ul>
    <li>Элемент списка 1</li>
    <li>Элемент списка 2</li>
</ul>

А вот пример ссылки:

<a href="https://example.com">Перейти на сайт</a>

Урок 4: Работа с изображениями

Изображения добавляются с помощью тега <img>. Пример использования:

<img src="image.jpg" alt="Описание изображения">

Атрибут alt описывает изображение и отображается, если изображение не удалось загрузить.

Урок 5: Таблицы

Таблицы используются для отображения данных в виде строк и столбцов. Пример таблицы:

<table>
    <tr>
        <th>Заголовок 1</th>
        <th>Заголовок 2</th>
    </tr>
    <tr>
        <td>Данные 1</td>
        <td>Данные 2</td>
    </tr>
</table>

Урок 6: Формы

Формы используются для сбора данных от пользователей. Пример формы:

<form action="/submit" method="post">
    <label for="name">Имя:</label>
    <input type="text" id="name" name="name">
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    <input type="submit" value="Отправить">
</form>

Урок 7: Мета-теги

Мета-теги используются для предоставления информации о веб-странице. Пример мета-тега:

<meta name="description" content="Описание страницы">

Урок 8: Встраивание видео

Видео можно встроить в страницу с помощью тега <video>. Пример:

<video controls>
    <source src="video.mp4" type="video/mp4">
    Ваш браузер не поддерживает видео.
</video>
